# Youtube Channel Website
A one page website for your YouTube channel.

## What to Change
- Background image in *styles.css*
- Text in <title> tag
- "Channel Name" in header
- Links in header
- Text in "About" box
- "Watch Now" link
- Embedded video ID
- Playlist link images
