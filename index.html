My Projects

YouTube

I run a technology based YouTube page and have reached over 30,000 unique viewers. I feel most passionate about this project more than any, as the feeling of educating others is very mentally rewarding. I look to continue with YouTube and grow my channel as much as possible.

Game Console Modding

I've been modding consoles for the past 3 years, maninly focusing on the Xbox 360 using the RGH and JTAG methods. I do this to bring life back into old consoles that might be considered outdated or useless. This can be anything from running backup games, to running Linux.

Coding

Although I'm not a professinal coder, I have done a few reasonable projects using JavaScript and HTML. One of my first projects was a cheating client for the popular online game, "Kahoot". I also created an RTE tool for the Xbox 360 copy of Skate 3, which allows you to change aspects of the game using custom offset editing.

PCB Design

I've designed multiple PCB's, one of which is used along with a popular project called "Wifiibo" which allows you to read and write Nintendo Amiibo NFC tags. My PCB board allows easy access to the soldering points and compacts them down into a slim design.
